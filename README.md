## Project Summary

This project develops a custom **YOLOv11 object detection model** for autonomous driving applications by fine-tuning a pre-trained model on a filtered subset of the COCO dataset. A custom dataset containing eight traffic-related object classes (person, car, truck, motorcycle, stop sign, traffic light, bus, and bicycle) is automatically generated by parsing COCO annotations, converting bounding boxes into YOLO format, and creating training and validation datasets.

The model is trained using **transfer learning** and evaluated against the original pre-trained YOLOv11 model using metrics including **Precision, Recall, F1 Score, mAP@50, mAP@50-95, IoU, FPS, and inference latency**. Performance is further analysed through confusion matrices, precision-recall curves, F1-confidence curves, IoU histograms, and training versus validation loss graphs.

Finally, the trained model is used to perform **object detection and object tracking** on both images and videos, demonstrating its ability to recognise and track multiple road users and traffic infrastructure in realistic driving scenarios.
